Introduction

In this assignment a Semantic Web application was build. The user can send sparql queries to the Stardog triple store, which will return data from the ontology that is uploaded on Stardog. The ontology used is about pasta dishes. In the first part the user can send sparql queries and results will be shown with inferencing, in the second part without inferencing. The use of inferencing will lead to more results, since these automatic procedures will generate new relationships.
